# ♿ Accesibilidad (WCAG) & Contraste

## Regla de Oro
**Borde siempre más oscuro/perceptible que el fondo.**

## Valores Mínimos en Modo Oscuro
- **Fondo:** opacidad mínima `0.04`
- **Borde:** opacidad mínima `0.10`
- **Separadores:** perceptibles (`rgba(255,255,255,0.08)`)

## Prohibiciones
- ❌ **PROHIBIDO** usar gris genérico como separador.
- ✅ Usar siempre tintes del acento correspondiente (ej. `#dde5ff` para blue).
