# 🧩 Elementos de UI

## 1. Cards KPI
- **Claro**: Gradiente 135deg `{lightBg}` a `#ffffff`. Borde `lightBorder`.
- **Oscuro**: Gradiente `rgba(accent, 0.18)` a `rgba(255,255,255, 0.02)`. Borde `rgba(accent, 0.40)`.

## 2. Botones de Acción (Filas)
- **Specs**: Padding `5px 10px`, `rounded-lg`, `11px semibold`.
- **Variantes**: Ver (Blue), Eliminar (Red), Aprobar (Emerald), Revisar (Yellow).

## 3. Badges de Estado
- **Specs**: `rounded-full`, `11px 600`, padding `3px 9px`.
- **Colores**: Según tabla en `migration-mapping.md`.
