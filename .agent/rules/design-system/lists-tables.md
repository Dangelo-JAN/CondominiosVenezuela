# 📋 Tablas y Listas (ThemedList)

## 1. Componentes Obligatorios
Uso de `ThemedListWrapper`, `ThemedHeadingBar` y `ThemedListContainer`.

## 2. Estilos de Fila
- **Separador Claro**: `#c5d4ec` (cvblue-200 aprox).
- **Separador Oscuro**: `rgba(255,255,255,0.08)`.
- **Hover**: 
  - Claro: `#e5ecf8` (Blue) / `#fffce8` (Yellow).
  - Oscuro: `rgba(accent, 0.07)`.
